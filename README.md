## How to run

1. Clone this repository
2. Install dependencies

```bash
npm i
```

3. Run mongodb with

```bash
mongod --dbpath ~/.mongo
```

4. set you .env file

5. Run the project with

```bash
npm run start
```

6. Run the project in development mode (nodemon)

```bash
npm run start:dev
```
